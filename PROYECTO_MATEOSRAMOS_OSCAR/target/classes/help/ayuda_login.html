Ayuda - Pantalla de Inicio de Sesión

Esta ventana permite acceder al sistema utilizando tu usuario y contraseña. A continuación encontrarás una guía rápida para entender cada elemento.

1. Introducir el usuario

Escribe tu nombre de usuario en el campo “Usuario”. Debe coincidir exactamente con el registrado en el sistema.

2. Introducir la contraseña

Escribe tu contraseña en el campo “Contraseña”. Si deseas verla, pulsa el icono del ojo.

3. Mostrar u ocultar contraseña

El icono del ojo permite alternar entre ver la contraseña en texto claro o mantenerla oculta.

4. Iniciar sesión

Pulsa el botón “Iniciar Sesión”. Si los datos son correctos, accederás al menú correspondiente a tu perfil (Administrador, Profesor o Estudiante).

5. Recuperar contraseña

Si no recuerdas tu contraseña, pulsa “Recuperar Contraseña”. El sistema te guiará en el proceso de recuperación.
Nota: Si aparece un mensaje en rojo, significa que el usuario o la contraseña no son correctos. Si aparece en verde, el inicio de sesión fue exitoso.
